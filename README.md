# PNN Atlas

> A **free** and **open-source** atlas for Perineuronal Nets in the mouse brain.
>

## Requirements

- Python 3.10
- Numpy 1.22.3  
`pip install numpy`
- matplotlib 3.5.2  
`pip install matplotlib`
- Pandas 1.4.2  
`pip install pandas`
- Dash 2.4.1  
`pip install dash`
- Dash Bootstrap Components 1.1.0  
`pip install dash-bootstrap-components`
